### 1. OR (|)
0 0 1 1    運算元1

0 1 0 1    運算元2

———-

0 1 1 1 (運算元1 | 運算元2) 

### 2. AND (&)
0 0 1 1    運算元1

0 1 0 1    運算元2

———-

0 0 0 1 (運算元1 & 運算元2) 

### 3. XOR (^)
0 0 1 1    運算元1

0 1 0 1    運算元2

———-

0 1 1 0 (運算元1 ^ 運算元2) 

---
性質

1. 交換律

2. 結合律（即(a^b)^c == a^(b^c)）

3. 對於任何數x，都有 x^x=0，x^0=x

4. 自反性 A ^ B ^ B = A ^ 0 = A

### 4. Reverse (~)
0 0 1 1    運算元1
———-

1 1 0 0 (~運算元1)
